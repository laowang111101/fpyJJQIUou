# 医院管理系统 python914

本项目为大学生Python课程设计中的数据库大作业，采用全栈开发模式，致力于实现稳定高效的医院管理系统。

## 技术栈

- **前端**：HTML + CSS + JavaScript
- **后端**：Flask
- **数据库**：SQL

## 功能模块

- 基础数据管理
  - 患者信息管理（增删改查）
  - 医生信息管理（增删改查）
  - 药品信息管理（增删改查）
- 系统管理
  - 管理员登录

## 系统角色

- 系统管理员
- 普通用户（医生、患者）

## 运行环境

- Python环境
- Flask框架
- SQL数据库

## 部署步骤

1. 安装Python环境
2. 安装Flask框架
3. 建立数据库，导入数据
4. 运行Flask应用
